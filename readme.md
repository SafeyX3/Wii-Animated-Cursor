### **what it do**

turns your cursor into the little wii grabby hand and when you click it grabs!!!!!!!



### **keybinds**

press alt + shift + esc to exit the program!!!!!!



### **credits**

i got the cursors from [https://www.rw-designer.com/cursor-set/wii](https://www.rw-designer.com/cursor-set/wii)


### **what the files actually are**
readme.md - self explanatory
wii-grab-ccw.cur - cursor file for the grabby hand (ccw stands for counter clock wise)
wii-open-ccw.cur - cursor file for the idle (open) hand
WiiCursor.ahk - the original ahk script for it
WiiCursor.exe - compiled (?) version of the ahk script
